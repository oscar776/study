#### 1、查看状态

```javascript
git status
```

#### 2、提交到暂存区

```javascript
git add 文件名.后缀
```

#### 3、把暂存区文件提交到分支

```
git commit -m '文件信息'
```

首次提交需要配置

```javascript
 git config --global user.email "you@example.com" 
 git config --global user.name "Your Name"  
```

#### 4、回滚

```
git reset  版本号
```
